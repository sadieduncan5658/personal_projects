**Actions:**

| Action                 | Windows / Linux    | macOS            |
| --------------------   | -------------------|----------------- |
| Delete terminal(s)     | Ctrl + Shift + W   | Cmd + Shift + W  |

## Git Commands
[Back to Top](#Keyboard-Shortcuts)
- ```git init``` - Initialize projects to use git
- ```git add .``` - Add all changes to be saved
- ```git add <filename>``` - Add a single file to be saved 
- ```git commit -m "<message>"``` - Save changes with a message 
- ```git push origin master``` - Push changes to GitHub 
- ```git push origin <branch>``` - Push changes to a specific branch 
- ```git pull origin master``` - Pull changes from GitHub master branch 
- ```git checkout -b <branch>``` - Create a new branch
- ```git status``` - Check status of changes 
- ```git branch``` - Check different branches 
- ```git log``` - See all previous changes
- ```git checkout <commit-#>``` - Travel back to an older commit
